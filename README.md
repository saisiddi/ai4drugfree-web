# AI4DRUGFREE - AI for Drug Free India 2026

AI4DRUGFREE is the official event site for the AI for Drug Free India 2026 movement, hosted by the School of Engineering and Technology. The experience highlights a multi-day program (8-14 May, 2026) focused on awareness, innovation, and community action for a drug-free future.

The site includes:

- A cinematic landing experience that introduces the movement and its goals.
- A curated event showcase with schedules, coordinators, and registration paths.
- A dedicated registration flow for each event track.

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion + Lenis for motion and smooth scrolling
- Three.js for the hero background

## Local Development

```bash
npm run dev
```

Open http://localhost:3000 in your browser.

## Production Build

```bash
npm run build
npm run start
```
